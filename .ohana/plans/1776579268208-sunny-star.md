# Plan: Mradi Africa Website Redesign

## Summary
Transform the Mradi Africa website from its current basic state to match the professional design template provided. Rebrand from "Forge Studio" to "Mradi Africa", showcase Dairy HQ and Hesabu as real case studies, migrate from pnpm to bun, and update dependencies.

## Phase 1: Package Manager Migration (pnpm → bun)
1. Remove `pnpm-lock.yaml` and `node_modules`
2. Remove `packageManager` field from `package.json`
3. Run `bun install` to generate `bun.lock`
4. Update scripts in `package.json` if needed

## Phase 2: Dependency Updates
- Update all devDependencies to latest compatible versions
- Verify build still works after updates

## Phase 3: CSS & Design System
- **`src/global.css`**: Add all template styles (nav, hero, projects, services, process, about, testimonials, contact, footer, modal, responsive breakpoints) — the CSS variables are already in place
- **`tailwind.config.js`**: Already configured with design tokens, keep as-is

## Phase 4: Component Extraction & Page Rebuild
Break the monolithic `index.tsx` into Qwik components under `src/components/`:

### Components to create:
1. **`src/components/nav/nav.tsx`** — Sticky nav with blur backdrop, logo, links, CTA, hamburger mobile menu
2. **`src/components/hero/hero.tsx`** — Eyebrow, headline, subtitle, CTA buttons, stat ticker
3. **`src/components/projects/projects.tsx`** — Project cards grid with case study data
4. **`src/components/projects/project-card.tsx`** — Individual project card
5. **`src/components/projects/case-study-modal.tsx`** — Modal overlay with full case study
6. **`src/components/services/services.tsx`** — 6 service cards in grid
7. **`src/components/process/process.tsx`** — 4 horizontal steps
8. **`src/components/about/about.tsx`** — About text + team grid
9. **`src/components/testimonials/testimonials.tsx`** — 3 testimonial cards
10. **`src/components/contact/contact.tsx`** — Dark section with info + form
11. **`src/components/footer/footer.tsx`** — Dark footer

### Layout changes:
- **`src/routes/layout.tsx`** — Keep minimal (Slot only), nav/footer in page

### Page:
- **`src/routes/index.tsx`** — Compose all components, Mradi Africa content

## Phase 5: Content Adaptation
Rebrand all content from Forge Studio → Mradi Africa:
- **Company**: Mradi Africa, Nairobi Kenya, hello@mradiafrica.com, +254 758 337 480
- **Tagline**: "We build products that scale with your ambitions" (adapted)
- **Case Studies**: Dairy HQ (featured), Hesabu Accounting, plus 4 other representative projects
- **Services**: Product Engineering, Platform & Infra, AI Integration, Design & UX, Code Audit, Technical Strategy
- **Team**: Use Mradi Africa team (placeholder names)
- **Testimonials**: Adapted testimonials referencing Dairy HQ and Hesabu
- **Stats**: Adapted to Mradi Africa context

## Phase 6: Build & Verify

2. Check for TypeScript errors
3. Test responsive breakpoints
